void EncryptWithPassword(Stream dataToEncrypt, string password, Stream destination)

Reads data from stream (parameter dataToEncrypt) and writes encrypted data to stream (parameter destination). Can be later decrypted using DecryptWithPassword. Salt and IV will be embedded to resulting stream. IV and salt are generated by CryptoRandom which is using System.Security.Cryptography.Rfc2898DeriveBytes. IV size is 16 bytes (128 bits) and key size will be 32 bytes (256 bits).
